# My Online Tools

Kumpulan online tools berbasis **Vanilla JavaScript** dan **PHP**.

## Tools Included
- Hashing Tools (SHA, bcrypt, dll)
- File Hash Generator
- DNS Tools
- IP Address Lookup
- Subnet Calculator
- QR Code Generator
- Image Compressor (JPG)
- Unit Converter
- Text Counter
- dll

## Tech Stack
- HTML, CSS
- Vanilla JavaScript
- PHP (for server-side processing)

## Notes
- PHP tools require a PHP-enabled web server
- This repository is source code only
